# Taxometer

Мультимовний React Native застосунок таксометра на Expo SDK 57 / React Native 0.86.

## Можливості

- тарифи з посадкою, включеними кілометрами, ціною за км, очікуванням і мінімальною сумою;
- GPS-вимірювання відстані та розрахунок вартості у foreground і background;
- пауза, завершення й локальна історія поїздок;
- підсумок та PDF-чек через системне меню поширення;
- українська, англійська та польська мови;
- локальна робота без облікового запису або сервера.

## Запуск

```bash
npm install
npx expo run:ios
```

Для Android використовуйте `npx expo run:android`, для браузера — `npm run web`. Після першого нативного build наступні запуски можна робити через `npm start` і встановлений Expo Dev Client.

Фонове відстеження потребує development/production build — Expo Go не підтримує background location повністю. Після початку поїздки надайте дозвіл геолокації «Завжди». Android показує системне сповіщення протягом активної поїздки. Якщо користувач примусово закриє застосунок, поведінка залежить від обмежень iOS/Android і виробника пристрою.

Дані поїздок зберігаються в AsyncStorage на пристрої.

## TestFlight

Релізний профіль описаний у `eas.json`. Перед першою збіркою зв’яжіть локальний проєкт з потрібним Expo/EAS owner:

```bash
npx eas-cli@latest init
npx eas-cli@latest build --platform ios --profile production
npx eas-cli@latest submit --platform ios --profile production
```

Для збірки та завантаження потрібні активний Apple Developer Program, запис застосунку в App Store Connect і Bundle ID `com.oleksiiburdeniuk.taxometer`. Детальний передрелізний чекліст міститься у `TESTFLIGHT.md`.

## Розрахунок тарифу

Застосунок використовує одинарний режим тарифікації `S`, описаний у EU MI-007 та OIML R 21:

- швидкість перемикання автоматично дорівнює `(ціна очікування за хв × 60) / ціна за км`;
- нижче цієї швидкості нараховується лише часовий тариф;
- вище цієї швидкості нараховується лише тариф за відстань;
- одна GPS-ділянка ніколи не оплачується одночасно за часом і відстанню;
- включена в посадку відстань використовується як спільний еквівалент початкової відстані або часу;
- GPS-точки з точністю гірше 50 м, нереалістичні стрибки та дрейф на місці не додаються до відстані;
- фінальна вартість округлюється до двох знаків після коми.

Це програмний GPS-таксометр, а не сертифікований законодавчо регульований вимірювальний прилад. Для комерційного використання перевірте місцеві вимоги до метрологічної сертифікації, тарифних кроків і джерела сигналу відстані.
